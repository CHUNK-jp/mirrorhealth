Features

なぜMirrorHealthか。

既存の健康アプリはあなたのデータをクラウドに送り、広告やサブスクで収益化します。MirrorHealthは違います。

🔒
完全ローカル処理
すべてのデータ処理はあなたのデバイス上で完結。外部サーバーへの送信はゼロ。ネット接続不要でも動作します。
🤖
ローカルAI分析
Ollama経由でLLMがあなたの健康トレンドを深く分析。歩数・睡眠・HRVの傾向から具体的なアドバイスを生成します。
📊
美しいダッシュボード
90日分の歩数・心拍・睡眠・HRV・消費カロリーをインタラクティブなチャートで可視化。7日平均も一目でわかります。
📱
Apple Health対応
iPhoneのヘルスケアアプリからエクスポートしたexport.xmlをドロップするだけ。インポート操作は一切不要。
🌐
日本語・英語対応
AIの分析レポートを日本語・英語どちらでも生成できます。ワンクリックで切り替え可能。
⚡
無料・オープンソース
MITライセンスで完全無料。サブスクリプションなし。広告なし。コードは全公開でカスタマイズ自由。
Setup

3ステップで起動。

Node.jsとOllamaがあれば、5分で動きます。

1
Ollamaをインストールしてモデルをダウンロード
ollama.com からインストール後、ターミナルで：
ollama pull llama3.2
2
リポジトリをクローンして起動
git clone https://github.com/CHUNK-jp/mirrorhealth.git
cd mirrorhealth
npm install
npm run dev
ブラウザで http://localhost:3000 を開く。
3
Apple Healthからデータをエクスポートしてドロップするだけ
iPhoneの「ヘルスケア」→ プロフィールアイコン → 「ヘルスケアデータを書き出す」。Zipのなかのexport.xmlをブラウザにドロップするだけです。
